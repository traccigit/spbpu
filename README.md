# spbpu

## 8 вариант АиСД СпбПУ ИКНК

Дан список владельцев телефонов в виде:


ФАМИЛИЯ ТЕЛЕФОН

15 симв. 10 симв.

## Пример входного файла:
Петров          9111634576

Фёдоров         9111635687

Отсортировать этот список в порядке убывания номеров телефонов, используя метод вставок. 

## Пример выходного файла:
Фёдоров         9111635687

Петров          9111634576
